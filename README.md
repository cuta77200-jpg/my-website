
# متجر البخور (تصميم مُحدّث)

قالب ثابت لبيع البخور والعود، يعمل محليًا في المتصفح دون خادم.

الميزات الأساسية:
- واجهة عربية مُحسّنة مع ألوان قهوية وبيضاء مناسبة لمتجر بخور.
- إدارة المنتجات عبر `admin.html` (العنوان، وصف، السعر، وصورة).
- السلة محفوظة في `localStorage` ويمكن تأكيد الطلب وإرساله عبر واتساب.

طريقة الاستخدام السريع:
1. افتح `admin.html` وأضف منتجات البخور: املأ العنوان، الوصف (اختياري)، السعر ورابط الصورة (اختياري).
2. ضع رقم واتساب المستلم في حقل "رقم واتساب المستلم" بصيغة دولية بدون + أو مسافات (مثال للسعودية: `9665XXXXXXXX`) ثم اضغط "حفظ الرقم".
3. افتح `index.html` للعرض والتسوق، اضغط "أضف إلى السلة" على المنتجات المطلوبة.
4. افتح `checkout.html`، أدخل اسم العميل ورقم الجوال ثم اضغط "تأكيد الحجز وإرسال عبر واتساب".

ملاحظات مهمة:
- تأكد من أن رقم واتساب المستلم بصيغة رقمية دولية بدون علامات (مثال: `9665...`).
- يعمل النظام بفتح رابط `https://wa.me/<seller>?text=...` لتعبئة رسالة الطلب في واتساب.
- هذا حل بسيط للتشغيل المحلي. إذا رغبت بإضافة استضافة، رفع صور أو قاعدة بيانات، أستطيع تنفيذ ذلك (مثال: Node.js + SQLite أو استخدام Firebase).

هل تريد أن أضيف:
- خدمة رفع الصور مع خادم بسيط؟
- استضافة المشروع على GitHub Pages أو نشره؟
- تحسينات إضافية في التصميم أو خيارات الدفع؟
